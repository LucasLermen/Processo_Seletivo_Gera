## Tecnologia

- [Angular](https://angular.io/)


## Rodando o projeto
Primeiramente instale as dependências:

```bash
npm install
```

Rode o servidor de desenvolvimento:

```bash
ng serve
```
Abra seu browser no [http://localhost:4200](http://localhost:4200) para ver o resultado.
